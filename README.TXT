A straightforward assembler that transforms RISC-V assembly code into binary machine code.
Features
Compatible with the RV32I instruction set.
Converts assembly into 32-bit RISC-V machine code.
Outputs binary files for simulation or execution.

How to Use
Write your RISC-V assembly code.
Execute the assembler to generate the corresponding machine code.
Use the generated binary with a simulator or emulator.
